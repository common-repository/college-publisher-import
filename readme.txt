== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-csv-importer` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to the CSV importer, select the file to upload and click on Upload.
